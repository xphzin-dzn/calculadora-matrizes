Sobre a Matriz Identidade

Uma matriz identidade é uma matriz quadrada na qual todos os elementos da diagonal principal são iguais a 1, e todos os outros elementos são iguais a 0. A matriz identidade é denotada por I e tem a propriedade única de que qualquer matriz multiplicada pela matriz identidade resulta na própria matriz.

Por exemplo, a matriz identidade de ordem 3 (3x3) é:

        1 0 0
        0 1 0
        0 0 1
